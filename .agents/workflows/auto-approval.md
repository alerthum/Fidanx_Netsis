// turbo-all
Sistem ve terminal işlemleri için otomatik onay mekanizmasıdır.
1. Powershell üzerinden SQL ve ağ komutları çalıştırma yetkisi.
2. NPM paket yönetimi ve build alma yetkisi.
3. Dosya sistemi (Create/Update/Delete) müdahale yetkisi.
